#为什么用jade
jade > html 
标签不会出现未闭合错误
block extands 模块化
可以循环减少重复代码

tip: 可以直接作为后端模版语言，expressjs 默认jade,python可以用pyjade 

------------------

#为什么用stylus
stylus > css
css prefix -webkit- -moz- "nib"库
整体设计自定义变量 font-color=#f2f2f2 
mixin extand 
stylus > sass > less 灵活
缺点 过于灵活导致项目换人接手会比较头疼，思考？
